# 内置样式

## 说明

V-Control组件功能的实现，并不依赖全局样式，全局样式只是提供一些基础且常用的样式，未指定Key属性的样式会覆盖您的全局样式。


全局样式的资源字典路径为
```
.\src\Resources\Styles\VControlGlobalStyles.xaml
```


## 默认按钮样式

![alt text](assets/image.png)

默认按钮样式提供了主题色背景和文字颜色。

同时提供以下几种按钮样式资源：

* PrimaryButton - 基本按钮
* TextButton - 文本按钮
* OutlinedButton - 线框按钮
* LightButton - 轻盈按钮（淡背景色）

默认按钮修改了禁用状态时的样式
